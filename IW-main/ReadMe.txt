Setup the python virtual enviroment using python -m venv /path/to/venv --system-site-packages

Each time, to start the venv, run 'source /path/to/venv/bin/activate'

Install the required python packages using pip install -r requirements.txt while using the venv


Compilation
javac -cp /home/PATH_TO_JEP/jep-4.2.1.jar/’*’:. /PathTo/Class.java

Execution
java -cp /home/PATH_TO_JEP/jep-4.2.1.jar/’*’:. Class [args]

Unzip the converted quantized model or use your own .tflite model/labels.txt.
